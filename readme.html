ToN Stats Tracker v1.1

VRChatのホラーゲーム「Terrors of Nowhere (ToN)」の戦績を自動で記録し、詳細に分析するためのツールセットです。

📋 必須コンポーネント

1. ToN Save Manager (VRChat側)
ゲーム内の情報を外部へ送信するために必須です。未導入の場合は以下から導入してください。
🔗 ToN Save Manager (GitHub)

🚀 使用手順

1. データの記録開始（プレイ前）
同梱の start_tracker.bat をダブルクリックして起動します。コンソールに「OSCパケットを待機しています...」と表示されたら準備完了です。
2. 分析ビューアーへのアクセス
記録されたデータを確認するには、以下のWebサイトにアクセスしてください。
🔗 ToN Statistics Viewer (Web版)
