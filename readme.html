ToN Stats Tracker v1.1

Build 20260309

VRChatの人気ホラーゲーム「Terrors of Nowhere (ToN)」における、あなたの全戦績を詳細に記録・可視化するための高度な補助ツールセットです。

📋 必須コンポーネント

1. ToN Save Manager (外部ツール)

VRChatのイベントを外部へ中継するために必須です。以下のリポジトリより最新版を導入してください。

🔗 ToN Save Manager (GitHub)
2. Windows PowerShell

OS標準のPowerShell 5.1以降が必要です。本ツールの記録プログラム（.ps1）を動作させます。

⚙️ 初期設定

  • VRChatを起動し、メニューの Settings > OSC を開き、Enabled に設定します。
  • 本ツールの全ファイルを同じフォルダに展開します。

🚀 使用方法

1. 記録の開始 (プレイ前)

start_tracker.bat をダブルクリックして起動します。コンソールに青文字で「OSCパケットを待機しています...」と表示されれば、記録待機状態です。

2. 統計の確認

以下の分析用Webサイトへアクセスし、本ツールフォルダ内の ton_stats.csv を読み込んでください。

🔗 ToN Statistics Viewer (Web)

❓ よくある質問 (FAQ)

Q. 記録が開始されません

A. VRChatのOSC設定が「Enabled」になっているか、および「ToN Save Manager」が正しく起動しているかを確認してください。

Q. 統計ツールで「全部敗北」になります

A. CSVファイルのカラム順やデータ形式が変更されていないか確認してください。最新版のビューアーを使用することを推奨します。
