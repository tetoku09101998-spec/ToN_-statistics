ToN Stats Tracker v1.0
VRChatのホラーゲーム「Terrors of Nowhere (ToN)」の戦績を自動で記録し、ブラウザで詳細に分析するための軽量ツールセットです。

📋 概要
  本ツールは、VRChatから送信されるOSCパケットをキャッチしてCSVに記録し、そのデータをブラウザ上で視覚化します。

同梱ファイル
  start_tracker.bat: ツール起動用バッチファイル（これをダブルクリックして開始）
  ton_tracker.ps1: 記録処理を行うプログラム本体
  ton_tracker.html: 記録されたデータをグラフ化する分析用ビューアー
  ton_stats.csv: 記録されたデータが蓄積されるファイル（初回起動時に自動生成）

⚙️ 事前準備
  VRChatの設定:
    VRChatを起動し、メニューの Settings > OSC タブを開きます。  
    OSCを「Enabled」 に設定してください。

OSの環境:
  Windows 10 / 11

🚀 使用手順
  1. データの記録（プレイ前）
    VRChatでToNをプレイする前に、start_tracker.bat をダブルクリックして起動します。
    黒い画面（コンソール）に「OSCパケットを待機しています...」と表示されれば準備完了です。
    そのままゲームをプレイしてください。ラウンドが終了するたびに、自動的に戦績が保存されます。
  
  2. データの分析（いつでも可能）
    ton_tracker.html をブラウザ（Chrome, Edge等）で開きます。
    画面上の「ファイルを選択」ボタンを押し、同じフォルダにある ton_stats.csv を読み込みます。
    「ダッシュボード」「チャート分析」「対戦履歴」のタブを切り替えて、自分の戦績を確認してください。

📊 主な分析機能
  プレイスタイル適性: レーダーチャートで「全体」「クラシック」「特殊」「単体」「複数」の勝率バランスを確認。
  ネメシス分析: テラー別の棒グラフをクリックすると、そのテラーとの詳細な因縁（勝率、最頻出マップ、最終遭遇日）を表示。
  詳細フィルタ: 特定の「ラウンド」「マップ」「テラー」を組み合わせて勝率を抽出可能。
