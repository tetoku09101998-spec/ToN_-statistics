VRChatの人気ホラーゲーム「Terrors of Nowhere (ToN)」における、戦績を詳細に記録・可視化するための高度な補助ツールセットです。

### 📖 マニュアル (Official Manual)
詳しい初期設定や使い方は、言語切り替えに対応した以下の公式マニュアル（Web版）を確認してください。

### 📋 必須コンポーネント
1. **ToN Save Manager**
   [GitHubリポジトリ](https://github.com/nomlasvrc/ToNSaveManager)より導入してください。
2. **Windows PowerShell**
   OS標準のPowerShell 5.1以降が必要です。

<h2 id="english">🇬🇧 English</h2>

An advanced auxiliary toolset for detailed recording and visualization of your complete battle records in VRChat's popular horror game "Terrors of Nowhere (ToN)".

### 📖 Official Manual
For detailed setup instructions and usage, please refer to the official web manual below (supports English and Japanese).
👉 [ToN Stats Tracker - Official Manual](https://tetoku09101998-spec.github.io/ToN_-statistics/manual.html)

*(※This link assumes the HTML file is uploaded to GitHub Pages as `manual.html`)*

### 📋 Requirements
1. **ToN Save Manager**
   Please install from the [GitHub Repository](https://github.com/nomlasvrc/ToNSaveManager).
2. **Windows PowerShell**
   Windows built-in PowerShell 5.1 or later is required.
